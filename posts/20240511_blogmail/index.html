blogmail

Date: May 11, 2024

blogmail is my own little blog engine, built in shy of 400 locs.

To steal eh.. reuse my code for your own project (shh.. it's GPL-3 licensed :-P), do the following:

go install github.com/4thel00z/blogmail/...@latest
git clone git@github.com:4thel00z/blackmail.dev.git
sed -i 's/blackmail.dev/coolstuff.dev/g' blackmail.dev/index.html
mv blackmail.dev coolstuff.dev
rm -rf coolstuff.dev/posts/*
echo "[]" > coolstuff.dev/posts.json
cd coolstuff.dev
blogmail create

Ok, you cool sock. Now go and enjoy your new blog :-)

Don't forget to host it on Github pages for free !
